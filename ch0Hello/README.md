## node 安装 typescript

npm install -g typescript

## 创建 tsconfig.json 文件

创建 tsconfig.json 文件

tsc --init

## 编译一个 TypeScript 文件

tsc hello.ts

## Vscode 编辑器及环境搭建

- Vscode 扩展中搜索 Prettier，并安装
- Vscode 设置中（settings）搜索 save，找到 Format On Save，勾选上，让代码在保存时自动格式化

## 安装 ts-node

npm install -g ts-node

运行： ts-node test.ts
